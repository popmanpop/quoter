## **Запуск сервиса**  

### **Windows**  
1. Перейдите в папку с проектом в терминале.  
2. Введите команду: ```quoter ```  

### **macOS / Linux**  
1. Перейдите в папку с проектом в терминале.  
2. Введите команду: ```go run./cmd/main.go ```  

После запуска сервис будет доступен по адресу:  `http://localhost:8080`
